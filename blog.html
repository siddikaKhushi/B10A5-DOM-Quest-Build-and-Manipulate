What is Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, allowing programming languages like JavaScript to interact with, manipulate, and update the content, structure, and style of a webpage dynamically.

What is event delegation in the context of DOM, and it is useful?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element to manage events on its child elements. Instead of attaching separate event listeners to individual child elements, the parent handles the event, taking advantage of the event bubbling mechanism in the DOM.
When an event occurs on an element, it bubbles up through its ancestors (parents, grandparents, etc.) in the DOM. By attaching a single event listener to a common ancestor, you can capture events from its children as they propagate upward. The parent can determine which child element triggered the event using the event.target property.
